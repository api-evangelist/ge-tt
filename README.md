# Ge.ttThis is a repo for managing machine readable artifacts for the [Ge.tt APIs](http://ge.tt). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Ge.tt APIs](http://ge.tt).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://ge.tt)- [Developer](http://ge.tt)- [Twitter](https://twitter.com/gettsharing)- [GitHub](https://github.com/gett)The [APIs.json](https://github.com/api-evangelist/ge-tt/blob/master/apis.json) index provides links to all the relevant links for the Ge.tt APIs, including the OpenAPI and Postman if they are present.
